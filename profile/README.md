## 핵심 기능

사진을 업로드하면 AI 기반으로 적절한 해시태그를 추천해주며, 추천 받은 해시태그를 이용해 사진을 소셜 미디어에 직접 공유할 수 있습니다.

## 주요 기능

- **사진 업로드 및 처리**: 사용자는 웹사이트를 통해 자신의 사진을 업로드할 수 있습니다.
